# Top-Down 2D Shooter

Fast paced **top-down 2D shooter** made using **Unity 2022.3.5f1** and **C#**.

## Features

 * Movement and shooting mechanics
 * Animations and sound
 * Variety of weapons
 * Different enemies

## Controls

| Action | Key / Input       |
| ------ | ----------------- |
| Move   | WASD              |
| Shoot  | Mouse             |
| Pause  | Esc               |

## Installation
 * Build yourself

 * Grab the latest release
 
 * Play online on the browser
