# Top-Down 2D Shooter

Fast paced **top-down 2D shooter** made using **Unity 2022.3.5f1** and **C#**.

## Features

 * Movement and shooting mechanics
 * Animations and sound
 * Variety of weapons
 * Different enemies

## Installation

### Open project
1. Clone the repository to a folder:

   ```bash
   git clone https://github.com/Mandom64/bestgame1.git
   ```
2. Open **Unity Hub**
3. Click **Add** and select the repository folder
4. Open the project in Unity
> **Note:** This project was developed using **Unity 2022.3.5f1**. Compatibility with other Unity versions has not been tested.
